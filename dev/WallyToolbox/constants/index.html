Constants

Configuration

The following constants are not exported by the toolbox; their use is mostly intended for developers and some particular use cases. If extending module data, please consider using the reference path.

WallyToolbox.WALLYTOOLBOXPATH — Constant

Path to supporting modules annex to WallyToolbox.

source
WallyToolbox.WALLYTOOLBOXDATA — Constant

Path to data files used by WallyToolbox.

source
WallyToolbox.MAJORDOMEPATH — Constant

Path to Python modules from legacy Majordome package.

source
